# Universal Asset Forge

Welcome to the `coreason-meta-engineering` Documentation. This repository serves as the **Air-gapped Electronic Design Automation (EDA) toolchain** for the CoReason ecosystem.

## Overview

Unlike passive data libraries, this repository is **Active by Design**. It acts as a kinetic toolchain for algorithmic code generation, built upon the strict paradigm of treating Python source code as an immutable abstract syntactic graph, not a simple flat file. 

The universal asset forge guarantees determinism, cryptographic canonical hashing of structures, and rigid isolation.

## Tripartite Fabrication Lines (MCP Tools)

The forge operates via three explicit fabrication lines to inject strictly validated AST logic into the source trees:

### [scaffold_manifest_state] (Passive Data Fab)
Generates standardized State Models bounded by cryptographic alignment axioms. Ensures pure deterministic canonicalization by inheriting from the macro `CoreasonBaseState`.

### [scaffold_logic_actuator] (Active Logic Fab)
Scaffolds deterministic logic actuators, properly equipped with typed MCP schemas and standard topological bounding logic. Includes the embedding of Uniform Resource Names (`actionSpaceId`).

### [scaffold_epistemic_node] (Autonomous Entity Fab)
Instantiates bounded Swarm/Agent entities, generating autonomous endpoints governed by zero-trust constraints and correct lifecycle hooks. Applies the epistemic docstring architecture.
