# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-04-17

### Added
- Genesis release of the Universal Asset Forge.
- Implementation of strict AST-based `libcst` code injection limits and idempotent boundaries.
- Establishment of the Tripartite Fabrication Lines (State, Actuator, Epistemic Node).
- Integration of Zero-Trust URN Semantic Validation and `actionSpaceId` publishing explicitly through `# MCP ROUTING TRIGGERS`.

### Changed
- Semantic refactoring to eliminate polysemantic legacy terminology (Model -> State, Tool -> Actuator, Agent -> Epistemic Node).
